# Aquaverse - Aquaculture AI Platform

Full-stack application for aquaculture disease diagnosis and management using AI.

## Project Structure

```
aquaverse-fullstack/
├── frontend/        # Next.js web application
├── backend/         # Node.js/Express API server
├── docs/            # Documentation
└── README.md        # This file
```

## Getting Started

### Prerequisites
- Node.js >= 20.0.0
- npm or yarn

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### Backend Setup
```bash
cd backend
npm install
npm run dev
```

## Features

- AI-powered prawn disease diagnosis
- User account management
- Real-time data analytics
- Responsive web interface

## Deployment

- **Frontend**: Vercel
- **Backend**: Render/Railway/Heroku (to be configured)
